What's included

  • Every .md file in the data directory, with the original folder layout.
  • Every attachment under _attachments/ (pasted images, drag-and-drop uploads).

What's NOT included

  • Admin config in .scrinium/ — user accounts (auth.db), the Flask session-signing key (secret.key), and category definitions (categories.json).
  • Hidden dotfiles anywhere in the data tree.

For full disaster recovery, also snapshot the .scrinium directory (or the entire data volume) outside of Scrinium — those files contain secrets and shouldn't be downloaded through the web UI.
